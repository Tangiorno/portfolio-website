Exige

Durant le 5ème semestre du BUT Informatique, étant en formation initiale, nous avions eu pour projet d'adapter sous la forme d'une interface web le jeu éducatif Exige développé il y a longtemps au sein de l'IUT.

Ce jeu consiste en une simulation de gestion d'entreprise, il a été programmé en BASIC, un langage obsolète aujourd'hui, et ne fonctionne plus sur des systèmes plus récents que Windows XP.

Le travail était donc d'analyser le code existant, puis de recréer l'algorithme sous la forme d'une application web.

Compétences développées :

  • Analyser du code en BASIC

  • Concevoir une application avec Vue.JS
