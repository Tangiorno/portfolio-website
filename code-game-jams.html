Code Game Jams

Nous avons participé avec mes camarades aux Code Game Jam 2023 et 2024, des concours de création de jeu vidéo en 30h.

La première fois, nous avons tenté de créer un jeu de plateformes intitulé Glove, sur le thème "Maître du temps". Ce projet fut un échec car nous avons perdu une grande partie des fichiers du jeu.

Notre deuxième participation fut plus fructueuse. Même si nous n'avons pas pu finir le jeu à temps, nous avons tout de même pu rendre une ébauche en partie jouable : un jeu de cartes intitulé Islands of Ether sur le thème "Légendes étherées".

Compétences mobilisées :

  • Développer un jeu vidéo avec Unity en C#

  • Organiser la création d'un jeu vidéo en équipe
